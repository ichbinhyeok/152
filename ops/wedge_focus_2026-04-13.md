# Wedge Focus - 2026-04-13

## The only wedge that matters right now
Do not start with:

- generic plumbing SEO
- broad gas safety education
- broad NYC landlord compliance blogging
- LMP directory pages
- borough pages with no verdict value

Start with:

- address or building-profile checker
- exact filing next step
- no-gas versus no-active-gas-service split
- extension, penalty, and waiver clarity
- LMP or filing-help routing

Support, but do not lead with:

- unsafe-condition explainers
- corrected-certification deep dives
- district pages
- challenge or exempt-building process pages

## Winning user state
The winning user is already in one of these states:

1. Needs to know if the building is due now
2. Knows the building is due but does not know the filing next step
3. Needs to distinguish no gas piping from no active gas service
4. Needs to know penalty, extension, or waiver exposure
5. Needs an LMP or certification helper quickly

## Pages that should win first
- `ll152 checker`
- `what do i file next ll152`
- `no gas piping vs no active gas service ll152`
- `ll152 extension penalty waiver`
- `2026 ll152 deadline`

Only after that wedge starts working:

- district overlays
- corrected certification pages
- unsafe condition pages
- exempt-building challenge pages

## Commercial focus for the first 4 to 6 weeks
Primary CTA order:

1. `Get LMP or filing help for this building`
2. `See the exact filing next step`
3. `Check penalty or waiver options`

Secondary CTA:

- `Check if this building is due now`
- `See the no-gas or no-active-gas path`

## Narrow operating loop
For the first implementation cycle, every build decision should support one loop:

1. User lands on checker or core route
2. User identifies building scope and gas status
3. Page returns applicability, next filing step, and deadline logic
4. Page explains penalty or waiver exposure if relevant
5. Page routes the user to the correct LMP or compliance-help form

## First traction signal
The first real traction signal is not traffic.

It is:

- repeated checker completions and clicks on `Get LMP or filing help`
- followed by at least a few qualified submissions tied to due-now or filing-next-step routes

## Operator memo
Do not try to out-blog plumbers. Win the moment when the owner wants one answer: `What exactly do I file next, and who handles it?`
